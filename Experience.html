Overview

Serinode is a 4G IoT data logger designed for sericulture farms, monitoring temperature, humidity, oxygen, CO2 levels, and transmitting data to InfluxDB for Grafana visualization and also enables local sd card storage. It includes an OLED screen for real-time readings and uses a Simcom A7670C GSM module.

Key Features

  • Environmental Monitoring: Tracks shed conditions.
  • Data Management: Stores data on InfluxDB and SD card.
  • User Interface: OLED screen for farmer access.
  • PCB Features:
    • Onboard Real-Time Clock circuit: Timestamps data.
    • Onboard SD Card circuit: Local storage.
    • Buck Converter circuit: Converts voltages up to 30V to 3.3V.
    • 4G gsm module:4G cellular connection capable
  • Enclosure Design: Custom-built for durability.
  • Dashboard Creation: Grafana for data visualization.
  • Independent Project: Designed and implemented by me.

Achievements and Challenges

From PCB design to Grafana setup, the project involved significant learning and problem-solving.As this was made by me from end to end ,i have had lot of exposure to real life experiences,ranging from pcb design limitaions,3d design considerations,product scalability,Remote connectivity limitations etc.
